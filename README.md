# Laboratory_work
Лабораторная работа №1: Работа с Git и GitHub

Описание проекта: Данный проект создан в рамках выполнения лабораторной работы №1. Цель работы: изучение платформы GitHub и практическое использование Git для хранения, обновления и распространения исходного кода проекта.

Задание по варианту: Вариант 1 — Геометрические фигуры.

Реализованы функции для вычисления параметров следующих фигур (распределены между участниками группы): Прямоугольник: периметр, площадь, длина диагонали;

Выполнил: Рожков Дмитрий — разработка модуля «Прямоугольник»; Реализованы функции для вычисления параметров следующих фигур (распределены между участниками группы): Ландин Дамир - Трапеция: периметр, площадь, длина средней линии.

Выполнил: Ландин Дамир - Трапеция: периметр, площадь, длина

Используемые технологии >> Язык программирования: C++

Система контроля версий: Git

Платформа для хостинга репозиториев: GitHub
